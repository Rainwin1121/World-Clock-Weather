🌍 Global Time & weather Monitor
是一个基于 Web 技术的单页应用 (SPA)，旨在为用户提供上帝视角般的全球信息监控体验。它摒弃了传统世界时钟的列表形式，将时间、天气、日照状态直接映射在深色模式的交互式地球地图上。无论您是跨国工作者、地理爱好者，还是需要一个炫酷的桌面副屏摆件，这个应用都能满足您的需求。
✨ 核心功能1. 🌐 全球节点监控 (Global Monitoring)核心枢纽：覆盖纽约、伦敦、东京、赫尔辛基等 10+ 个世界核心城市，以半透明毛玻璃卡片形式常驻显示。次级网络：覆盖欧洲、亚洲、非洲、美洲的 30+ 个重要节点（如巴黎、开罗、曼谷），以呼吸光点形式呈现，保持地图清爽。
2. 🤖 智能感知与交互 (Smart Intelligence)自动定位：利用 Geolocation API 自动识别您的位置，并在地图上生成高亮的 "MY LOCATION" 专属卡片。智能缩放：当您放大地图查看细节时，城市卡片会自动放大并增强显示层级，提供更清晰的阅读体验。全能搜索：内置 Open-Meteo 地理编码搜索，支持输入英文城市名（如 "Beijing", "Oslo"），自动定位并加载全球任意城市的数据。
3. ☀️ 深度气象数据 (Deep Weather)实时天气：显示气温、天气状况（自动区分昼/夜图标）。日出日落：精确计算当地的日出与日落时间。15天趋势：点击任意城市，展开视图，查看未来 15 天的气温趋势。AI 穿衣助手：根据当前气温和天气代码，智能生成出行与穿衣建议（如“寒冷，建议穿羽绒服”或“有雨，别忘带伞”）。
4. 📱 极致体验 (User Experience)深色模式：专为 OLED 屏幕和暗光环境优化，省电且护眼。PWA 支持：可直接添加到手机或平板桌面，像原生 App 一样全屏运行（无浏览器地址栏）。响应式设计：适配从显示器到手机的所有屏幕尺寸。
